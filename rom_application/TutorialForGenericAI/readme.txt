This directory contains the necessary files to run a tutorial on using the ROMApp for different kinds of AI tasks.
Specifically, parameters input and snapshots retrieval are demondtrated. Also residual and jacobian retrieval for Discrete PINN-like aplication.

This tutorial uses Pytorch and ScikitLearn, so make sure they are installed in your environment.

Open file TutorialForGenericAI.ipynb to read the tutorial, or download the directory to modify and run it yourself.
